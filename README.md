## 🌟 为什么选这个合集？

1. **双维度分类**：按「办公场景\+上手难度」分类，不用在海量工具中筛选，Ctrl\+F就能快速找到需要的工具，节省时间。

2. **信息无冗余**：每款工具仅标注核心信息——核心功能、免费额度、上手难度（★1\-5星）、适用人群、官方链接，不堆砌无关内容。

3. **免费优先**：优先收录免费工具、开源工具，或有明确免费额度的工具，拒绝“看似免费、实则付费”的套路工具。

4. **持续迭代**：每周更新1\-2次，新增热门工具、删除失效工具，同步工具免费额度变化，保证内容不过时。

5. **社区驱动**：欢迎所有人提交PR补充工具，新手也能轻松参与，共同维护高质量办公工具库。

## 🛠️ 使用指南

1. **快速查找**：按「Ctrl\+F」（Windows）/「Command\+F」（Mac），输入工具名称或功能（如“PPT”“OCR”），即可快速定位到对应工具。

2. **工具选择**：同一场景有多款工具时，优先选择「上手难度低\+免费额度高」的工具（标注★1\-2星的），新手无需纠结，直接试用。

3. **避坑技巧**：使用工具前，先查看「免费额度」，避免误触付费功能；部分工具需要注册账号，优先用邮箱注册，无需绑定过多信息。

4. **更新提醒**：每周日会更新工具列表，新增热门工具、删除失效工具，可Star项目持续关注更新。

4. 避免重复：提交前先搜索工具名称，确认合集内没有该工具，避免重复提交。
